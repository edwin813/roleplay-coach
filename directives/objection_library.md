# Directive: Objection Library

## Purpose
Define the company-approved objection scenarios and correct responses that agents must use during training. This is the "answer key" that the AI uses to evaluate agent performance.

## Structure
Each objection includes:
- **Objection Type**: Category (confusion, resistance, logistics, timing, cost)
- **Customer Statement**: What the AI "customer" will say
- **Correct Response Elements**: Key points the agent MUST include
- **Scoring Weight**: How important this objection is (1-10)
- **Difficulty Level**: Beginner, Intermediate, or Advanced

---

## Objection 1: Confusion - What Is This? What's the Benefit Package?

**Type**: Confusion
**Difficulty**: Beginner
**Scoring Weight**: 9/10

**Customer Statement**:
"What is this again? What's the benefit package?"

**Correct Response Elements** (Agent must hit at least 3 of 4):
1. **Identify the benefits**: "These are the Benefits that [sponsor] set up for you"
2. **Mention privacy**: "These benefits are private" and explain they can only be reviewed in the secure portal
3. **Explain sponsor relationship**: Mention that [sponsor] is a [Veteran/policyholder/credit union member]
4. **Clarify extension**: "[Sponsor] was able to extend that access to you"

**If they ask again, use secondary response**:
"These are the Benefits that [sponsor] set up for you. These include private life and health benefits, we will be going over it in detail in the secure portal, just like we do with our [veterans, union members, policy holders]"

**Common Mistakes to Avoid**:
- Don't skip mentioning the sponsor's name
- Don't fail to emphasize the private/secure nature
- Don't go into benefit details before getting to the portal

**Scoring**:
- 4/4 elements = 10 points
- 3/4 elements = 8 points
- 2/4 elements = 5 points
- 1/4 elements = 2 points
- 0/4 elements = 0 points

## Objection 2: Already Have Coverage - We Already Have Life Insurance

**Type**: Coverage
**Difficulty**: Beginner
**Scoring Weight**: 8/10

**Customer Statement**:
"We already have life insurance."

**Correct Response Elements** (Agent must hit at least 3 of 4):
1. **Acknowledge perfectly**: "That's exactly why I'm calling"
2. **Reference sponsor**: "[Sponsor] mentioned you might already have something in place"
3. **Position as complementary**: "These benefits are specifically designed to complement the insurance policies that you already have"
4. **Logic statement**: "This wouldn't even make sense for you if you didn't already have something set up"

**Common Mistakes to Avoid**:
- Don't act surprised they have insurance
- Don't immediately try to replace their current insurance
- Don't fail to position as complementary

**Scoring**:
- 4/4 elements = 10 points
- 3/4 elements = 8 points
- 2/4 elements = 5 points
- 1/4 elements = 2 points
- 0/4 elements = 0 points

## Objection 3: Resistance - I'm Not Interested

**Type**: Resistance
**Difficulty**: Intermediate
**Scoring Weight**: 10/10

**Customer Statement**:
"I'm not interested."

**Correct Response Elements** (Agent must hit ALL 4):
1. **Address confusion**: "Oh there might be some confusion"
2. **Remind about sponsor**: "Again [sponsor] included you in his family benefits"
3. **Reframe**: "There isn't anything to be interested in, [sponsor] already set it for you"
4. **State obligation**: "I just promised [sponsor] I would go over it with you"

**Common Mistakes to Avoid**:
- Don't accept "not interested" and end call
- Don't get defensive or pushy
- Don't fail to use the sponsor's name

**Scoring**:
- 4/4 elements = 10 points
- 3/4 elements = 6 points
- 2/4 elements = 3 points
- 0-1/4 elements = 0 points

## Objection 4: Logistics - I Didn't Get a Text Message

**Type**: Logistics
**Difficulty**: Beginner
**Scoring Weight**: 6/10

**Customer Statement**:
"I didn't get a text message."

**Correct Response Elements** (Agent must hit at least 3 of 4):
1. **Offer to resend**: "Ok let me resend it from the group text with [their sponsor]"
2. **Confirm number**: "Do you receive text to this number?"
3. **Action statement**: "Ok, great I just resent it"
4. **Get confirmation**: "Let me know when you receive it?" (confirm they receive it)

**Common Mistakes to Avoid**:
- Don't move forward without confirming they got the text
- Don't blame the customer
- Don't skip verification

**Scoring**:
- 4/4 elements = 10 points
- 3/4 elements = 8 points
- 2/4 elements = 4 points
- 0-1/4 elements = 0 points

## Objection 5: Timing - How Long Will This Take?

**Type**: Timing
**Difficulty**: Beginner
**Scoring Weight**: 7/10

**Customer Statement**:
"How long will this take?"

**Correct Response Elements** (Agent must hit at least 2 of 3):
1. **Acknowledge question**: "Great question"
2. **Set time expectation**: "I told [sponsor] I would set aside 30-45 minutes for you, in case you have any questions"
3. **Explain purpose**: "So we will be going over all of the private benefits [sponsor] set up for you during this time"

**Common Mistakes to Avoid**:
- Don't lowball the time estimate
- Don't make it sound optional
- Don't fail to explain what happens during that time

**Scoring**:
- 3/3 elements = 10 points
- 2/3 elements = 7 points
- 1/3 elements = 3 points
- 0/3 elements = 0 points

## Objection 6: Cost - How Much Will This Cost? Does This Cost Money?

**Type**: Cost
**Difficulty**: Intermediate
**Scoring Weight**: 10/10

**Customer Statement**:
"How much will this cost?" OR "Does this cost money?"

**Correct Response Elements** (Agent must hit at least 5 of 7):
1. **Reference text message**: "You must have not read the text message that [Sponsor] sent you"
2. **Free policy**: "I am going to be issuing you a $2000 policy for free"
3. **Needs analysis**: "In addition we will complete a no cost needs analysis for your family"
4. **Conditional options**: "IF we determine that there is a need and IF you're able to qualify I can show you some options that you can purchase"
5. **No promises**: "BUT I don't even know if you'll qualify so I can't make any promises"
6. **Confirm free policy**: "One thing I can promise is that we'll issue you the free $2000 policy"
7. **Portal requirement**: "But we can only do that in the secure benefits portal"

**Common Mistakes to Avoid**:
- Don't immediately talk about costs
- Don't skip mentioning the free $2000 policy
- Don't fail to mention qualification requirements

**Scoring**:
- 7/7 elements = 10 points
- 5-6/7 elements = 8 points
- 4/7 elements = 6 points
- 2-3/7 elements = 3 points
- 0-1/7 elements = 0 points

## Objection 7: Coverage Confusion - Is This Insurance? I'm Already Covered

**Type**: Coverage
**Difficulty**: Intermediate
**Scoring Weight**: 9/10

**Customer Statement**:
"Is this insurance?" OR "I'm already covered."

**Correct Response Elements** (Agent must hit at least 4 of 5):
1. **Ask if they talked to sponsor**: "Did you talk to [sponsor]?"
2. **Mention thousands in benefits**: "Ok they set you up with thousands of dollars in family benefits"
3. **Explain private access**: "He/she also wanted me to make sure you had access to these private benefits"
4. **Qualification requirement**: "YOU DO have to qualify for these"
5. **Fallback benefit**: "Even if you cannot qualify I will still be able to give you a no cost benefits package as a thank you from [sponsor]"

**Common Mistakes to Avoid**:
- Don't avoid the word "insurance" if they ask directly
- Don't skip mentioning qualification
- Don't fail to reference the sponsor

**Scoring**:
- 5/5 elements = 10 points
- 4/5 elements = 8 points
- 3/5 elements = 6 points
- 2/5 elements = 3 points
- 0-1/5 elements = 0 points

## Objection 9: Too Busy - I'm Busy Right Now, Call Me Back

**Type**: Busy
**Difficulty**: Advanced
**Scoring Weight**: 9/10

**Customer Statement**:
"Look, I'm really busy right now. Can you just call me back some other time?"

**Correct Response Elements** (Agent must hit at least 5 of 7):
1. **Acknowledge**: "I totally understand, I'm too busy to go over all of this right now"
2. **Pivot to scheduling**: "I'm just calling to schedule a time that will work for you"
3. **Reframe purpose**: "We are going to issue and activate ALL of the benefits that [sponsor] set up for you"
4. **Offer time options**: "When are you and [spouse name] most available? Mornings, afternoons, or evenings?"
5. **Give two specific slots**: "My schedule is getting really full, but I can get this taken care of for you at either [time] or [time]. Which one works best?"
6. **Text confirmation**: "Can you be sure to text back and respond with the word 'CONFIRM' so we know you received it?"
7. **Commitment close**: "We do ask that if you book the appointment, to commit to the appointment. We mostly handle benefits for US veterans, police, and union members. If you miss your appointment, that would be time we could be servicing another family."

**Common Mistakes to Avoid**:
- Don't agree to "call back later" without scheduling a specific time
- Don't give more than two time options
- Don't forget to mention both lead and spouse need to be present
- Don't forget the "CONFIRM" text request
- Don't say "thank you" at the end

**Scoring**:
- 7/7 elements = 10 points
- 5-6/7 elements = 8 points
- 3-4/7 elements = 5 points
- 1-2/7 elements = 2 points
- 0/7 elements = 0 points

## Usage Notes

- The AI will randomly select 3-5 objections per session based on difficulty level
- Scoring is weighted - harder objections (higher weight) contribute more to final score
- System tracks which objections agents struggle with most and reports to trainer
- Update this library as you discover new common objections or refine responses
- All responses MUST reference the sponsor by name when indicated
- The "[sponsor]" placeholder will be replaced with the actual sponsor's name during training sessions
